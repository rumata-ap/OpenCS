# OpenCS

**OpenCS** — open-source десктопное приложение для расчёта и проверки строительных конструкций по российским нормативным документам (СП, ГОСТ).

Реализован полный цикл: от ввода геометрии сечения, задания материалов и нагрузок до расчёта прочности, трещиностойкости, огнестойкости, кручения и теплопередачи. Включает собственный МКЭ-решатель для рам и оболочек с интеграцией расчёта сечений.

---

## Возможности

### Железобетонные стержневые сечения (СП 63.13330)

- **Фибровый метод** — деление сечения на волокна (прямоугольная сетка, триангуляция Рупперта, метод продвижения фронта)
- **Контурный метод** — интегрирование по теореме Грина для полигональных областей без фибр
- Расчёт прочности (I группа предельных состояний): поиск предельного коэффициента нагружения, проверка по предельным деформациям сжатого бетона (ε_b2, ε_b0) и растянутой арматуры (ε_s,ult)
- Расчёт трещиностойкости (II группа предельных состояний): ширина раскрытия трещин (acrc)
- Потери предварительного напряжения по СП 63
- Двухстадийные сечения для сборно-монолитных конструкций и усиления

### Железобетонные плитные/оболочечные сечения (СП 63.13330)

- Кинематика Кирхгофа — Лява (6 компонент: мембранные + изгибные)
- Слоистая модель с послойным интегрированием бетона и арматурных слоёв
- Прочность (ULS): деформационный критерий п. 8.1.30, Wood-Armer, Caprani-Capurso, полная слоистая модель
- Трещиностойкость (SLS): расчёт ширины раскрытия трещин по п. 8.2.15–8.2.18
- Непродолжительное и продолжительное раскрытие

### Стальные сечения (СП 16.13330.2017)

- Проверки прочности (раздел 8): центральное сжатие/растяжение, изгиб, сжатие с изгибом, растяжение с изгибом, поперечный изгиб (срез), кручение, изгиб с кручением, продольный изгиб, местное смятие стенки
- Проверки устойчивости: местная устойчивость стенки и полки
- Конструктивные проверки
- Классификация сечений по п. 7.3
- Автоматическое вычисление геометрических характеристик (A, I, W)

### Комбинации нагрузок (СП 20.13330)

- Основное и особое сочетания
- Типы нагрузок: постоянная (G), длительная (L), кратковременная (Q), особая (A)
- Автоматический разбор имён наборов, пользовательские γ_f
- Огибающая (Envelope) и список комбинаций (Cases)
- Для стержней: [N, Mx, My, Vx, Vy, T]
- Для пластин: [Nx, Ny, Nxy, Mx, My, Mxy, Qx, Qy]

### Расчёт кручения (Сен-Венан)

- Два метода: метод граничных элементов (МГЭ, порт TORSCON) и метод конечных элементов (МКЭ, функция Прандтля)
- Результаты: постоянная кручения It, координаты центра кручения, поле касательных напряжений, максимальное напряжение
- Поддержка линейных (Tri3) и квадратичных (Tri6) конечных элементов

### Теплопередача

- Нестационарная и стационарная теплопередача (уравнение теплопроводности)
- Тепловые граничные условия: конвекция, заданная температура
- Поддержка произвольных тепловых материалов
- Квадратичные тепловые элементы, итерационный решатель Пикара

### Огнестойкость (СП 468, ГОСТ 30247.0)

- Огневые кривые: ISO 834, Hydrocarbon (EN 1991-1-2), Slow Heat
- R-проверки предела огнестойкости по несущей способности
- Фибровый метод: поэлементная γ-редукция с температурным полем
- MVP-метод (максимальных значений параметров)
- Температурная редукция бетона (силикатный/карбонатный/лёгкий заполнитель) и арматуры по СП 468

### МКЭ-расчёты

- Линейный статический анализ рам (2D/3D) и оболочек
- Элементы: балка Эйлера — Бернулли, оболочки MITC3/MITC4
- Интеграция с расчётом сечений через адаптер CSfea.CScore
- Разреженные матрицы (собственная библиотека + обёртка CSparse)
- Импорт расчётных схем из ЛираСАПР (CSV и COM API)

### Импорт/экспорт

- DXF (netDxf) и AutoCAD COM API — импорт геометрии сечений
- ЛираСАПР CSV и COM API — импорт схем и усилий (ЗН/РСН/РСУ)
- CSV — экспорт результатов
- Excel — импорт .xls (ExcelDataReader), экспорт .xlsx (MiniExcel)
- SQLite — персистентность всех данных проекта

## Структура решения

```
OpenCS.sln
├── CSmath/                 Чистая математика (векторы, матрицы, сплайны, геометрия)
├── CSTriangulation/        Триангуляция (Advancing Front, Ruppert CDT)
├── CScore/                 Доменная модель и расчётный движок (fibre method)
│   ├── CrossSection.cs     Стержневое сечение
│   ├── PlateSection.cs     Плитное сечение
│   ├── MaterialArea.cs     Фибровое разбиение и интегрирование
│   ├── Steel/              Стальные сечения (СП 16)
│   ├── Combinations/       Комбинации нагрузок (СП 20)
│   └── Fire/               Огнестойкость (СП 468)
├── CScore.Fire/            Огнестойкость (надстройка над CScore)
├── CSfea.Sparse/           Разреженные матрицы
├── CSfea.Sparse.CSparse/   Обёртка CSparse (interop)
├── CSfea.Core/             МКЭ-ядро (балки, оболочки, решатели)
├── CSfea.CScore/           Мост CScore ↔ CSfea (адаптация единиц, осей)
├── CSfea.Thermal/          Теплопередача (МКЭ)
├── CSfea.Torsion/          Расчёт кручения (МГЭ/МКЭ)
├── CSfea.Tests/            Тесты МКЭ/кручения
├── OpenCS/                 WPF-приложение (точка входа)
│   ├── ViewModels/         ViewModel'и
│   ├── Views/              WPF-контролы и кастомная отрисовка
│   ├── Tasks/              Расчётные задачи
│   ├── Services/           Сервисы (файлы, диалоги, логи)
│   └── Utilites/           DatabaseService, утилиты
└── TriTest/                Песочница триангуляции/кручения
```

## Сборка и запуск

Требования: [.NET 9.0 SDK](https://dotnet.microsoft.com/download/dotnet/9.0).

```bash
dotnet build OpenCS.sln
dotnet run --project OpenCS
```

Сборка под Windows (WPF). Проекты без WPF-зависимостей (`CS*`) можно собрать и под другие платформы.

---

## Системные требования

- ОС: Windows 10/11
- .NET 9.0 Runtime
- Для импорта DXF: не требуется дополнительного ПО
- Для импорта из AutoCAD (COM API): установленный AutoCAD

---

## Назначение

OpenCS предназначен для инженеров-проектировщиков, научных сотрудников и студентов строительных специальностей. Программа позволяет выполнять поверочные расчёты существующих и проектируемых конструкций в соответствии с актуальными российскими нормами.
