# Объединение переключателей сетки FEM — дизайн

## Цель

Объединить в 3D-виде FEM две кнопки — простую сетку КЭ и сетку стержней — в один переключатель `Сетка`.

## Текущее состояние

`FemSchemaView3D` содержит два независимых `ToggleButton`:

- `showShellEdgesCheck` управляет `_shellEdgesVisual` — рёбрами сетки оболочек;
- `showMeshCheck` управляет `_meshVisual` и `_meshNodeGlyphVisual` — расчётной сеткой стержней и её узловыми глифами.

При пересборке 3D-визуалов `BuildVisuals()` очищает viewport, а затем отдельно добавляет каждый слой в зависимости от состояния соответствующей кнопки.

## Предлагаемое поведение

Оставить одну кнопку простой сетки с начальным состоянием `True`. Её локализованный tooltip должен называться `Сетка`.

При включённой кнопке отображаются все доступные сеточные слои:

1. `_shellEdgesVisual`, если присутствует;
2. `_meshVisual`, если присутствует;
3. `_meshNodeGlyphVisual`, если присутствует.

При выключенной кнопке все три слоя удаляются из viewport. Если какой-либо слой отсутствует в текущей модели, операция остаётся безопасной и не меняет остальные визуалы.

Кнопка `Сетка стержней` удаляется. Отдельного состояния для сетки стержней не вводится: её видимость полностью определяется общей кнопкой `Сетка`.

## Реализация

- Сохранить текущий toggle простой сетки и его начальное значение `True`.
- Объединить обработчики добавления/удаления сеточных слоёв в одном обработчике общей кнопки.
- Вызывать общий метод из `BuildVisuals()`, чтобы сетка стержней не исчезала после обновления данных или пересборки визуалов.
- Удалить отдельные XAML-элемент, обработчик и публичный метод, относящиеся только к `showMeshCheck`, если они больше не нужны.
- Обновить оба словаря ресурсов для текста `Сетка`; пользовательские строки в XAML должны оставаться через `DynamicResource`.
- Не изменять прочие режимы отображения: узлы, нагрузки, закрепления, оболочки, диаграммы и редактирование схемы.
- Сохранить существующие незакоммиченные изменения в `FemSchemaView3D.xaml`, не связанные с объединением кнопок.

## Проверка

- Добавить/обновить тест доступной логики, чтобы общая сетка добавляла и удаляла все три типа визуальных слоёв, включая частично отсутствующие слои.
- Проверить сборку `OpenCS.sln`.
- Проверить существующие FEM/OpenSees тесты.
- Выполнить визуальную проверку 3D-вида: сетка включена по умолчанию, одним переключателем скрываются/показываются и оболочечная сетка, и сетка стержней.
