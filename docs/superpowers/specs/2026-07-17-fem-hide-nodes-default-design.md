# Скрытие узлов по умолчанию в 3D-просмотре FEM-схемы

## Цель

Уменьшить время построения 3D-просмотра больших импортированных схем ЛИРА. В режиме редактирования OpenCS создаёт отдельный визуальный прокси для каждого узла; на крупных схемах это может блокировать UI.

## Дизайн

- Флажок «Узлы» в `FemSchemaView3D` выключен по умолчанию.
- При выключенном флажке узловые визуальные объекты и узловые прокси не создаются.
- При включении флажка 3D-просмотр перестраивается и показывает узлы; при выключении они удаляются.
- Отображение стержней, пластин и прокси стержней не меняется.
- В режиме обычного просмотра сохраняется существующий путь через `PointsVisual3D`; в режиме редактирования переключение управляет узловыми прокси.

## Проверка

- Добавить регрессионную проверку поведения переключателя на уровне доступной UI-логики либо выполнить ручную проверку после сборки.
- Выполнить сборку решения и существующие тесты `CScore.Tests` и `OpenCS.OpenSees.Tests`.

## Ограничения

Изменение не ограничивает число узлов в модели и не меняет данные импорта; оно только откладывает их визуализацию до явного включения пользователем.
